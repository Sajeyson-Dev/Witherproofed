# Blocks: 

### - Heavy Reinforced Obsidian - Base Block. Wither immune.
### - Heavy Reinforced Glass - Wither immune, has Ghostly variant.
### - Heavy Reinforced Translucent Glass - Same as normal Reinforced Glass, but translucent.
